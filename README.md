# Homomorphic Encryption for Privacy-Preserving Face Recognition

A secure face recognition system that uses homomorphic encryption to protect sensitive biometric data during processing.

## Features
- Privacy-preserving face recognition
- Encrypted biometric data processing
- Secure authentication workflow
- Database integration

## Technologies Used
- Python
- Flask
- Homomorphic Encryption
- SQLite

## Installation

pip install -r requirements.txt

python app.py
